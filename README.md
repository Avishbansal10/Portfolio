A highly polished, responsive, and modern personal portfolio website built with clean frontend architecture. Designed with a sleek slate-dark theme, featuring custom layouts for technical skills, professional work experience, cybersecurity hackathons, and full-stack projects.

🔗 **Live Link**: [https://portfoliobyavi.netlify.app/](https://portfoliobyavi.netlify.app/)

---

## ✨ Features

- **Premium Slate-Dark Theme**: Clean visual hierarchy utilizing unified spacing, subtle card hover scale animations, and custom vector icons.
- **Dynamic Projects Grid**: Showcases featured development projects (Full-Stack, APIs, Data Analytics) with direct links.
- **Timeline Columns**: Dedicated layouts for **Work Experience** (National Informatics Centre, Unified Mentors) and **Cyber Security Achievements** (C-DAC BOSS OS Bug Bounty).
- **Direct Credentials Verification**: Integrated links to view local PDF credentials for resumes and internship certifications.
- **Fully Responsive**: Optimized for seamless navigation across mobile, tablet, and widescreen desktop layouts.

---

## 🛠️ Technologies Used

- **Markup**: HTML5 (Semantic Structure)
- **Styling**: CSS3 (Modern Flexbox/Grid systems, Custom Variables, Transitions)
- **Typography & Icons**: Poppins (Google Fonts), FontAwesome 6.5.2
- **Hosting**: Netlify (Automatic Webpack CI/CD Pipeline)

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Avishbansal10/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in any modern web browser to view the site instantly.
